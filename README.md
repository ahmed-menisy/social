🛠 Tech Stack
Frontend Framework: Angular (v17+) / TypeScript

Styling: Tailwind CSS & Flowbite (Custom UI System)

State Management: Signals / RxJS

Performance: Server-Side Rendering (SSR) & Partial Hydration

Tooling: ESLint, Prettier, Husky (Git Hooks)

✨ Key Features
Authentication & Security: Secure User Login/Signup utilizing JWT and Route Guards for protected views.

Dynamic Feed: Real-time post updates with infinite scrolling and optimized image lazy loading.

User Interactions: Full CRUD operations for posts, likes, and nested commenting systems.

Profile Management: Customizable user profiles with dynamic image uploading and bio updates.

Responsive Design: Mobile-first approach using Tailwind's utility-first classes, ensuring a seamless experience from 4K monitors to smartphones.

🏗 Architectural Highlights
As a senior developer, I focused on several core engineering principles for this project:

Facade Pattern: Decoupling the UI components from the business logic and API services for better maintainability.

Component Composition: Building reusable, atomic UI components using Flowbite for consistent design language.

Performance Optimization: Achieving high Core Web Vitals through optimized Change Detection strategies and modular lazy loading.
